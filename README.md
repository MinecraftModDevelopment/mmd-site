# The source code for the Minecraft Mod Development modding group website along with documentation for their mods.

To get started once you have cloned this repo, open it up in your IDE of choice and run ``npm install`` in the terminal window and let that finish.

Once setup is complete you can use ``npm run dev`` in the same terminal window to run a local copy of the site to test with.

This site is published and managed by KiriCattus, for questions and issues please contact them via Discord
